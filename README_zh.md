# MIDI 工具集

> [English](README.md)

一个纯前端、零依赖的 Web MIDI 文本编辑器与音频导出工具集。使用纯律（Just Intonation）频率比编写音乐，支持实时播放、波形调节和 WAV/MIDI 导出。

---

## 快速开始

直接用浏览器打开 `index.html` 即可使用，无需安装任何依赖或构建工具。

```

```

## 功能一览

### MIDI 编辑器

以纯文本格式编写音符，从上到下依次演奏。每行一个音符，逗号分隔四个字段：

| 字段 | 格式 | 说明 | 默认值 |
|------|------|------|--------|
| 时值 | `分子/分母` | `1/1` = 四分音符，`1/2` = 八分音符，`1/8` = 三十二分音符 | `1/1` |
| 音高 | `分子/分母` | 纯律频率比，`1/1` = 基准音，`4/3` = 纯四度，`3/2` = 纯五度 | `1/1` |
| 大小 | `01` ~ `10` | 两位数字，控制力度 | `10` |
| 编号 | `00` ~ `99` | 两位数字，标记用，不影响演奏 | 空 |

示例：

```
1/1,1/1,10,01
1/1,9/8,10,02
1/1,5/4,10,03
1/1,4/3,10,04
1/1,3/2,10,05
1/1,5/3,10,06
1/1,15/8,10,07
1/1,2/1,10,08
```

支持 `//` 和 `#` 开头的注释行。

#### 编辑器功能

- **实时解析** — 输入时即时解析，错误行号标红，行号区即时更新
- **播放/暂停/停止** — 支持暂停后从上次位置继续播放
- **BPM 调节** — 20~400 BPM，支持滑块和数字输入
- **速度倍率** — 0.25x~4x，提供 0.5x / 1x / 1.5x / 2x 快捷预设
- **基准频率** — 默认 261.63 Hz（中央 C），可自由调整
- **音量控制** — 0~100%
- **波形选择** — 三角波 / 正弦波 / 方波 / 锯齿波
- **ADSR 包络** — 每个音符带 attack、decay、sustain、release 包络
- **泛音层** — 每个音符叠加一个 2 倍频正弦波泛音
- **播放高亮** — 演奏时当前行和行号高亮显示
- **音符信息** — 右侧面板实时显示当前音符的时值、音高、频率、力度、编号
- **示例模板** — 内置纯律音阶、小星星旋律、和弦进行示例
- **导入/导出** — 支持导入 `.txt` 谱面文件，导出为 `.txt`
- **自动保存** — 编辑器内容自动保存到 localStorage

#### 快捷键

| 按键 | 功能 |
|------|------|
| `Space` | 播放 / 暂停 |
| `Escape` | 停止 |
| `Ctrl+S` | 导出谱面文件 |
| `Tab` | 插入缩进 |

### 音频导出

切换到"音频导出"标签页，将编辑器中的音符导出为音频文件。

#### WAV 导出

- 使用 OfflineAudioContext 离线渲染，完全复现编辑器中的音色
- 支持 44100 / 48000 / 96000 Hz 采样率
- 支持 16-bit / 24-bit 位深度
- 保留波形设置、ADSR 包络和泛音叠加

#### MIDI 导出

- 将纯律频率比映射到最近的标准 MIDI 音符编号
- 自动使用原声大钢琴（Program 0）音色
- 音符时长支持两种模式：
  - **实际时值** — 按谱面中定义的时值精确映射
  - **固定八分音符** — 所有音符统一为八分音符长度
- 标准 MIDI 格式 0，PPQN = 480

#### 导出预览

- 实时显示音符数、总时长（拍）、有效 BPM、预计时长（秒）
- 预览音符列表，切换到该标签页时自动刷新

## 技术架构

### 浏览器兼容性

- 需要支持 Web Audio API（所有现代浏览器均支持）
- 推荐 Chrome / Edge / Firefox 最新版

### 核心模块

| 模块 | 文件 | 职责 |
|------|------|------|
| 标签切换 | `app.js` (L1-31) | 标签页导航与状态联动 |
| MIDI 编辑器引擎 | `app.js` (L33-590) | 解析、播放、暂停、UI 更新 |
| 音频导出工具 | `app.js` (L592-898) | WAV 编码、MIDI 生成、离线渲染 |
| 样式表 | `style.css` | 深色主题、响应式布局 |

### 音频引擎架构

```
Editor Input
    │
    ▼
parseEditor() ── 解析文本，生成 parsedNotes[]
    │
    ▼
play() ── 通过 AudioContext 调度振荡器
    │
    ├── Oscillator (主音色) ── Gain (ADSR) ──┐
    │                                         ├── destination
    ├── Oscillator (2倍频泛音) ── Gain ──────┘
    │
    ▼
scheduleUIUpdates() ── requestAnimationFrame 驱动播放高亮
```

## 设计哲学

- **纯律优先** — 音高使用分数比表示，不绑定十二平均律，支持任意律制
- **文本即乐谱** — 可读可写，易于版本控制、分享和批量编辑
- **零依赖** — 纯 HTML/CSS/JS，不依赖任何框架或构建工具
- **离线可用** — 所有资源本地化，无需网络连接
